# Tiles polite-poll fix

**Headline:** v1.8.3 dramatically cuts POTACAT's polling load on the Tiles on the Air spots endpoint. The operator (KK4ODA) reported on 2026-06-02 that aggregate POTACAT traffic was averaging 2–3 sustained req/sec against their Supabase Free tier Edge Function quota — enough to put their project into a grace-period warning that would have shut the API off entirely on June 9. v1.8.3 implements all three fixes they asked for so this doesn't recur.

## What changed

- **30-second base cadence** (was 20 s). Per the operator: anything between 10 and 30 seconds catches a new spot well within its 30-minute lifetime. Per-instance worst case drops from 3 req/min to 2 req/min.
- **Incremental polling via `since`** — POTACAT now tracks the timestamp of the newest spot it's seen and asks the server only for rows posted after that. Most response bodies are now near-empty instead of carrying the full snapshot every time, which is what makes the aggregate quota math work. A full snapshot is fetched every 10 polls (~5 minutes) to catch anything missed during a brief network blip and to age stale rows out of the local cache.
- **HTTP 429 Retry-After honored** — when Tiles' edge throttle returns 429, the desktop now reads the `Retry-After` header and pauses ALL Tiles polling until the indicated deadline. The cached spot list stays visible during the pause; the user sees no break in UI. Log line: `[Tiles] rate-limited; pausing polls for <N>s (server Retry-After honored)`.

Internally, `lib/tiles.js` exports a new `TilesRateLimitError` class with a `retryAfter` field so callers can distinguish "we're throttled" from "the API is down."

## Carries forward from v1.8.2

All v1.8.2 features and fixes — the redesigned Settings dashboard, multi-operator profiles, Cloud Tunnel orphan-adoption, Share Your Rig dialog, the `/api/pair` whitelist that unblocks pairing over the tunnel, the `_handleMessage` `req` crash fix, the EADDRINUSE retry — are all in v1.8.3 unchanged. See the v1.8.2 release notes for the full list.
