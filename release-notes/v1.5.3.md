A focused follow-up to v1.5.2: the **Bandspread popout launcher fix** is the headline, plus a handful of logging, CW, and rig-control improvements.

## The Bandspread Fix

- **Bandspread popout window now launches.** `preload-bandspread-popout.js` was missing from the packaged ASAR, so the popout silently failed to open on v1.5.2 installs. Now included in the build. If Bandspread wasn't working for you after updating yesterday, this is the fix.

## CW

- **ECHOCAT phone now runs its own iambic keyer** — CW sidetone on the phone is zero-latency, keyed directly in-browser. Remote CW through a spotty connection no longer sounds jittery.
- **CW paddle menu-setup hint** — when DTR keying is in use, POTACAT now logs an actionable hint pointing at the exact menu items you'd need to flip on the radio if your paddle isn't keying as expected.

## Rig Control

- **Split only (re)asserts on state change** — sending a fresh SPLIT ON command every polling cycle was stepping on manual split adjustments, and SPLIT OFF was being ignored entirely. Both fixed. Turning split off now actually turns split off.
- **SSB-over-DATA Yaesu hint** — new in-app warning for Yaesu users when their DATA MODE is set to **PSK** instead of **OTHR/USER**: SSB-over-DATA traffic gets mangled by the PSK modulation path. A one-time nudge the first time we detect it.

## Logging + ADIF

- **Activation park is logged (`MY_SIG_INFO`) even when you switch to Hunter mode mid-session.** Roving activators who take a hunter break no longer lose the park ref on their activation contacts.
- **Logger32 is now a supported logbook target** alongside Log4OM 2, HRD, N3FJP, and DXKeeper.
- **ADIF export remembers the last-used folder** — export a second batch and the file picker opens where you left off.

## Data

- **`cty.dat` CRLF-truncated keys fixed** — the last entry per block in `cty.dat` was dropping its final character on Windows because of Windows CRLF line endings. A handful of DX prefixes are now resolved correctly that weren't before.

---

**macOS users on Apple Silicon:** the DMG is ad-hoc signed (no paid Developer ID yet). If you see "POTACAT is damaged and can't be opened," run this once from Terminal and then relaunch:

```
sudo xattr -dr com.apple.quarantine /Applications/POTACAT.app
```

**Intel Mac users:** build from source for now — a universal DMG is coming in a later release.
