# Flex multiFlex TX with SmartSDR open, full rig-control parity on the phone, and APF

**Headline:** v1.8.18 lets you **transmit on a FlexRadio whether SmartSDR is open or closed** — no more flipping an audio/rig setting each time. It also **unifies rig control** so every DSP button (NR, ANF, Comp, VOX, AGC, filter) works and reports state correctly from **ECHOCAT on your phone**, adds an **Audio Peak Filter (APF)** control, and ships a **13 Colonies** event watchlist. ECHOCAT for **Android** is now live on Google Play.

## 📻 Transmit with SmartSDR open or closed (Flex multiFlex)

On SmartSDR Direct, having SmartSDR open used to block POTACAT from transmitting — you had to change an audio or rig setting every time you opened or closed it. POTACAT now **self-hosts as a second multiFlex GUI client** when SmartSDR is running, so you can:

- Run POTACAT with SmartSDR **open**, and keep operating after you **close** it — no setting changes either way.
- TX FT8 (JTCAT) cleanly while SmartSDR shows the panadapter.

Under the hood: POTACAT **auto-picks a free DAX channel** (so it never fights SmartSDR over one), **tunes its own slice** via the radio API instead of leaking onto SmartSDR's slice, keeps its DAX feed **unmuted** in multiFlex while still letting you mute the radio's front-panel speaker in SmartSDR, and shows the correct frequency. Requires **multiFlex enabled** on the radio (2 GUI clients).

> If you only ever run SmartSDR-less, nothing changes for you. multiFlex self-hosting kicks in only when SmartSDR is also connected.

## 🎛️ Full rig-control parity from ECHOCAT

Several DSP controls worked from the desktop but did nothing — or showed the wrong state — when sent from the **ECHOCAT** phone app: **NR**, **ANF**, **Comp**, and **VOX** didn't take, **AGC** didn't display its selection, and the **filter** control did nothing. All transports now route through a **single rig-control dispatcher** with a canonical control registry, so the phone and desktop can't drift apart again (guarded by a new parity test). NR/ANF/Comp/VOX now actuate and report correctly, AGC shows its mode, and the filter command works.

## 🔉 Audio Peak Filter (APF)

New **APF** control for FlexRadio — toggles the slice's CW Audio Peak Filter from both POTACAT and ECHOCAT.

## 🇺🇸 13 Colonies event watchlist

For the **13 Colonies** special event: one button (**Add to Watchlist**) loads the **K2A–K2M** stations plus the bonus stations so they star in your spot list, and you can **dismiss the countdown banner** if you don't want to see it in the run-up to the event.

## 📱 ECHOCAT on Android

The **ECHOCAT** remote app is now live on **Google Play** — the link in Settings is active alongside the App Store badge.

## 🐞 Fixes & polish

- **ECHOCAT:** a remote PTT-release no longer truncates an engine-owned **FT8 transmission**.
- **JTCAT:** chase-target picker now includes **BOTA (WWBOTA)** and **TILE (Tiles)** sources.
- **Diagnostics:** rig-control activity is captured in the mobile diagnostic snapshot for easier remote troubleshooting.
- **Build:** fixed the native **FT8 decoder build on macOS 26.5.1**.
- **Cloud:** the summary card now shows **Active** for paid members during their trial window.
