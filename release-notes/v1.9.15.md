CW text-send now actually keys on Yaesu rigs that key through a CW Key Port, finishing the fix started in 1.9.14. Also: the America 250 WAS banner now shows every state active in the same week.

## CW text-send keys again on the FT-891 / FT-710

1.9.14 stopped CW text-send from crashing the app on these rigs, but it still didn't key — because the underlying fault was a code error on that path that ran before any keying happened. That's now fixed: CW text sent to an FT-891 or FT-710 through its CW Key Port keys the rig for real.

If you're setting one of these up: select the exact model (FT-891 or FT-710) in Settings, wire the radio's second USB COM port as the CW Key Port, set the radio menu PC KEYING to DTR (or RTS, matched by Settings, Rig, CW keying line), and enable break-in.

## America 250 WAS banner shows all of the week's states

Some weeks of the ARRL America 250 Worked All States event run two or more states at once. The banner was only showing the first one — so a week with, say, Indiana and Utah only ever named Indiana. It now lists every state active that week, each paired with its own W1AW callsign (for example, "Indiana (W1AW/9) & Utah (W1AW/7)"). The states board already highlighted all of them; the banner now matches.
